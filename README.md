# Chrome 英译中翻译助手

一个简洁高效的Chrome浏览器插件,可以快速将网页上选中的英文文本翻译成中文。

## ✨ 功能特点

- 🎯 **即选即译**: 选中英文文本后自动显示翻译按钮
- 🖱️ **右键菜单**: 支持通过右键菜单触发翻译
- 🎨 **精美UI**: 现代化设计,渐变色彩,流畅动画
- ⚡ **快速响应**: 使用Google Translate API,翻译速度快
- 💾 **智能缓存**: 相同文本自动缓存,避免重复请求
- 🌐 **全网支持**: 在任何网页上都可以使用

## 📦 安装方法

### 开发者模式安装(本地)

1. 打开Chrome浏览器,访问 `chrome://extensions/`
2. 开启右上角的"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择本插件的文件夹 `chrome-translator`
5. 安装完成!

## 🚀 使用方法

### 方法一: 自动翻译按钮
1. 在任何网页上选中英文文本
2. 会自动显示紫色的"翻译"按钮
3. 点击按钮即可查看中文翻译

### 方法二: 右键菜单
1. 选中英文文本
2. 右键点击选中的文本
3. 选择"翻译成中文"
4. 查看翻译结果

## 🛠️ 技术栈

- **Manifest V3**: 使用最新的Chrome扩展API
- **Vanilla JavaScript**: 无依赖,纯原生JS实现
- **Google Translate API**: 免费翻译接口
- **Modern CSS**: 渐变、动画、响应式设计

## 🎨 设计亮点

- **渐变配色**: 紫色渐变(#667eea → #764ba2)
- **流畅动画**: 按钮和弹窗都有平滑的过渡效果
- **阴影效果**: 使用box-shadow增强视觉层次
- **响应式设计**: 适配不同网页环境

## 📝 注意事项

- 需要网络连接才能使用翻译功能
- 翻译结果由Google Translate提供
- 建议翻译文本长度不超过5000字符

## 🔧 开发说明

如需修改或扩展功能:

1. **更换翻译API**: 修改 `translator.js` 中的 `translateWithGoogle` 方法
2. **调整样式**: 编辑 `content.css` 文件
3. **修改触发方式**: 调整 `content.js` 中的事件监听
4. **添加新功能**: 在 `manifest.json` 中声明新权限
