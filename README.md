# Chrome 英译中翻译助手

一个简洁高效的Chrome浏览器插件,可以快速将网页上选中的英文文本翻译成中文。支持Google翻译和GPT翻译两种引擎。

## ✨ 功能特点

- 🎯 **即选即译**: 选中英文文本后自动翻译,无需额外操作
- 🤖 **双引擎支持**: 支持Google翻译(免费)和GPT翻译(高质量)
- 🖱️ **右键菜单**: 支持通过右键菜单触发翻译
- 📄 **段落/全文翻译**: 支持翻译整段或整个网页
- 🎨 **精美UI**: 现代化设计,渐变色彩,流畅动画
- ⚡ **快速响应**: 翻译速度快,体验流畅
- 💾 **智能缓存**: 相同文本自动缓存,避免重复请求
- ⏱️ **智能弹窗**: 翻译结果自动显示120秒,鼠标悬停可暂停
- 🌐 **全网支持**: 在任何网页上都可以使用

## 📦 安装方法

### 开发者模式安装(本地)

1. 打开Chrome浏览器,访问 `chrome://extensions/`
2. 开启右上角的"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择本插件的文件夹 `chrome-translator`
5. 安装完成!

## ⚙️ 配置翻译引擎

### 打开设置页面

1. 右键点击浏览器工具栏中的插件图标
2. 选择"选项"或"设置"
3. 或者访问 `chrome://extensions/`,找到本插件,点击"详细信息",然后点击"扩展程序选项"

### 选择翻译引擎

#### Google 翻译(默认,免费)
- ✅ 完全免费,无需配置
- ✅ 速度快,稳定可靠
- ⚠️ 翻译质量中等

#### GPT 翻译(需要API密钥)
- ✅ 翻译质量高,更自然流畅
- ✅ 支持多种GPT模型(GPT-3.5/GPT-4)
- ⚠️ 需要OpenAI API密钥
- ⚠️ 会产生API调用费用

### 配置GPT翻译

1. 访问 [OpenAI平台](https://platform.openai.com/api-keys) 获取API密钥
2. 在设置页面选择"GPT 翻译"
3. 输入您的API密钥(格式: sk-...)
4. 选择GPT模型:
   - **GPT-3.5 Turbo**(推荐): 性价比高,速度快
   - **GPT-4**: 质量最高,但费用较高
   - **GPT-4 Turbo**: 平衡质量和速度
5. (可选)如果使用第三方API代理,可修改API地址
6. 点击"保存设置"

## 🚀 使用方法

### 方法一: 自动翻译(推荐)
1. 在任何网页上选中英文文本
2. 自动显示翻译结果弹窗
3. 翻译结果会在120秒后自动关闭,或点击关闭按钮手动关闭

### 方法二: 右键菜单
1. 选中英文文本
2. 右键点击选中的文本
3. 选择"翻译成中文"
4. 查看翻译结果

### 方法三: 段落翻译
1. 点击页面右侧的悬浮工具栏
2. 点击"段落翻译"
3. 点击要翻译的段落
4. 查看翻译结果

### 方法四: 全文翻译
1. 点击页面右侧的悬浮工具栏
2. 点击"全文翻译"
3. 等待翻译完成
4. 查看完整翻译结果

## 🛠️ 技术栈

- **Manifest V3**: 使用最新的Chrome扩展API
- **Vanilla JavaScript**: 无依赖,纯原生JS实现
- **Google Translate API**: 免费翻译接口
- **OpenAI API**: GPT翻译接口
- **Chrome Storage API**: 配置持久化存储
- **Modern CSS**: 渐变、动画、响应式设计

## 🎨 设计亮点

- **阴影效果**: 使用box-shadow增强视觉层次

- **智能交互**: 避免误触,提供更好的用户体验

## 📝 注意事项

- 需要网络连接才能使用翻译功能
- Google翻译完全免费,无需配置
- GPT翻译需要OpenAI API密钥,会产生费用
- 建议先使用Google翻译测试功能
- 翻译结果会缓存,相同文本不会重复请求
- 在翻译弹窗内选择文本不会触发新的翻译

## 💰 费用说明

### Google 翻译
- 完全免费,无任何费用

### GPT 翻译
- 按OpenAI API调用计费
- GPT-3.5 Turbo: 约 $0.002/1K tokens
- GPT-4: 约 $0.03/1K tokens
- 一般短文本翻译费用很低(每次几分钱)

## 🔧 开发说明

如需修改或扩展功能:

1. **添加新的翻译引擎**: 在 `translator.js` 中添加新方法
2. **调整样式**: 编辑 `content.css` 文件
3. **修改触发方式**: 调整 `content.js` 中的事件监听
4. **添加新功能**: 在 `manifest.json` 中声明新权限
5. **修改配置选项**: 编辑 `options.html` 和 `options.js`
