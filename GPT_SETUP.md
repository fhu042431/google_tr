# GPT翻译配置指南

## 快速开始

### 1. 获取OpenAI API密钥

1. 访问 [OpenAI平台](https://platform.openai.com/api-keys)
2. 登录或注册OpenAI账号
3. 点击 "Create new secret key"
4. 复制生成的API密钥(格式: sk-...)
5. **重要**: 妥善保管密钥,不要分享给他人

### 2. 配置扩展

1. 在Chrome浏览器中,右键点击扩展图标
2. 选择"选项"打开设置页面
3. 选择"GPT 翻译"单选按钮
4. 粘贴您的API密钥
5. 选择GPT模型(推荐使用GPT-3.5 Turbo)
6. 点击"保存设置"

### 3. 开始使用

配置完成后,选择网页上的英文文本,即可使用GPT进行翻译!

## 模型选择建议

### GPT-3.5 Turbo (推荐)
- **优点**: 速度快,费用低,质量好
- **适用**: 日常翻译,大部分场景
- **费用**: 约 $0.002/1K tokens

### GPT-4
- **优点**: 翻译质量最高,理解能力强
- **适用**: 专业文档,复杂内容
- **费用**: 约 $0.03/1K tokens

### GPT-4 Turbo
- **优点**: 平衡速度和质量
- **适用**: 需要高质量但又关注成本
- **费用**: 约 $0.01/1K tokens

## 费用估算

一般情况下:
- 短句(10-20词): < $0.001
- 段落(100-200词): $0.001-0.005
- 长文(500-1000词): $0.01-0.03

**提示**: 扩展会自动缓存翻译结果,相同文本不会重复计费。

## 使用第三方API代理

如果您使用第三方OpenAI API代理服务:

1. 在设置页面找到"API 地址"选项
2. 输入代理服务提供的API地址
3. 确保API密钥与代理服务匹配
4. 保存设置

常见代理格式:
- `https://your-proxy.com/v1/chat/completions`
- `https://api.your-service.com/openai/v1/chat/completions`

## 常见问题

### Q: 为什么翻译失败?
A: 检查以下几点:
1. API密钥是否正确
2. OpenAI账户是否有余额
3. 网络连接是否正常
4. API地址是否正确

### Q: 如何查看API使用量?
A: 访问 [OpenAI使用情况](https://platform.openai.com/usage) 查看详细的API调用记录和费用。

### Q: GPT翻译和Google翻译有什么区别?
A: 
- **Google翻译**: 免费,速度快,适合日常使用
- **GPT翻译**: 收费,质量高,更自然流畅,适合专业翻译

### Q: 可以随时切换翻译引擎吗?
A: 可以!在设置页面随时切换,无需重启浏览器。

### Q: API密钥安全吗?
A: API密钥保存在Chrome的同步存储中,只有您的浏览器可以访问。建议:
- 不要在公共电脑上保存API密钥
- 定期更换API密钥
- 设置OpenAI API使用限额

## 技术支持

如有问题,请检查:
1. Chrome开发者工具的Console(F12)查看错误信息
2. 确认manifest.json中的权限配置正确
3. 确认网络可以访问OpenAI API
